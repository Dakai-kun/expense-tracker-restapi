REST API - Dokumentasi

Ringkasan endpoint yang tersedia pada aplikasi (menggabungkan `src/app.js` dan `src/index.js`).

Umum

Method GET
URL /
Output JSON: { message: 'REST API Express + Prisma berjalan' }

Categories (dari `app.js`)

Method GET / POST
URL /categories
GET Mengembalikan daftar kategori (sorted by name)
POST Body JSON: { name } — membuat kategori baru

Users (dari `app.js`)

Method GET / POST
URL /users
GET Mengembalikan semua pengguna
POST Body JSON: { name, email, photoUrl } — membuat atau update user (upsert)

Transactions (dari `app.js`)

Method GET / POST
URL /transactions
GET Opsional header: Authorization: <email|id> — jika diberikan, hanya transaksi user tersebut; include category & user
GET (detail) /transactions/:id — mengembalikan transaksi spesifik
POST Form multipart (field: title, categoryId, amount, type, date, imageId) atau file `image` — Authorization header wajib

Posts (dari `index.js`)

Method GET / POST / PUT / DELETE
URL /posts
GET Mengembalikan daftar post (ordered by createdAt desc)
GET (detail) /posts/:id — mengembalikan satu post
POST Body JSON: { title, content, published } — membuat post (title wajib)
PUT /posts/:id — Body JSON: { title, content, published } — memperbarui post
DELETE /posts/:id — menghapus post

Halaman dokumentasi ini adalah halaman statis. Untuk melihatnya setelah deploy ke Vercel, buka /docs pada URL aplikasi Anda.
